# desktop-dictionary
A small PowerShell script that lets me double-click a word and instantly see its meaning in a Windows notification using a shortcut.

1. Tools used: AutoHotkey (hotkeys & text capture), PowerShell (API calls), BurntToast (Windows toast notifications)
2. dictionary api used: https://api.dictionaryapi.dev/api/v2/entries/en/ 

## Setup Instructions:
1. Install AutoHotkey
2. clone repo
3. Install Burnt Toast
4. check img url in powershell file
5. Run the AutoHotkey script
6. select any text and type ALT+SHIFT+D (modify shortcut if you want in ahk file)
7. voila! notification recieved
